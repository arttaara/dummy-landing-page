## ğŸŒ Deployment (GitHub Pages)

This site is ready to be hosted for free on GitHub Pages.

1.  Upload all files to a new GitHub Repository.
2.  Go to **Settings** > **Pages**.
3.  Under **Build and deployment**, select **Source** -> `Deploy from a branch`.
4.  Select `main` (or `master`) branch and save.
5.  Wait 1-2 minutes, and GitHub will provide your live URL.

## ğŸ“ Notes for the Client

- **Images:** Currently using placeholders (`placehold.co`). You will need to replace the `src="..."` links in the HTML files with your actual image files (e.g., `images/painting1.jpg`).
